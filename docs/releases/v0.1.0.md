# VOIDcore v0.1.0

First public split checkpoint of VOIDcore as a standalone repository.

## Highlights

- fresh standalone `VoidCore` repository
- package/runtime rename to `io.aeyer.voidcore`
- Docker Compose stack boots in one shot
- bundled `Cityline MUD` door sidecar included in the default stack
- GPL-3.0-or-later licensing added
- first post-split cleanup of the public surface from instance-specific naming

## Included services

- `postgres`
- `app`
- `cityline-door`

## Verification

- full Gradle test suite passes
- `docker compose build app cityline-door`
- `docker compose up --build`

## Notes

This release is the public-engine checkpoint, not the final extraction state.
Some internal compatibility paths still refer to legacy "release" document
types while the user-facing surface has started moving toward the more generic
"files" model.
